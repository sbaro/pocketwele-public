## 🎮 Nos productions

### pocketWele · _Disponible · 2026_

**L'Awalé de poche, vif et coloré.**

Le grand classique africain du semis-capture, repensé en version joyeuse et
lumineuse. Jouez contre l'ordinateur (3 niveaux de difficulté) ou à deux sur le
même appareil.

|---|---|
| **Genre** | Jeu de plateau (Awalé / Oware) |
| **Plateformes** | iPhone · iPad · macOS |
| **Modes** | Solo (vs ordinateur) & 2 joueurs (pass-and-play) |
| **Statut** | Disponible |
